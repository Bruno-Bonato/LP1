8) Cada struct Pessoa ocupa 72 bytes no arquivo: são 4 bytes para cada variável do tipo int (idade e peso) e 64 bytes reservados para o array nome. Isso significa que, mesmo que o nome armazenado tenha poucos caracteres, o espaço completo de 64 bytes é sempre reservado. Por causa disso, o arquivo acaba ficando maior do que o necessário, ele guarda sempre o tamanho fixo da struct, independentemente do tamanho real do nome.
